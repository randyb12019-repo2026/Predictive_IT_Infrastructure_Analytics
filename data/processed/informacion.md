
# Data Processed

## Descripción

Esta carpeta contiene los datos resultantes del proceso de limpieza, transformación y enriquecimiento realizado sobre los datos originales.

## Transformaciones realizadas

* Creación de variables derivadas.
* Normalización de métricas numéricas.
* Clasificación por niveles de utilización y presión del sistema.
* Generación de etiquetas descriptivas para facilitar el análisis.

## Características

* Los archivos almacenados en esta carpeta están preparados para análisis exploratorio y modelado predictivo.
* Conservan la información relevante del dataset original junto con las nuevas variables generadas.

## Uso

Estos datos son utilizados durante las fases de análisis exploratorio, generación de insights y entrenamiento de modelos de Machine Learning.
