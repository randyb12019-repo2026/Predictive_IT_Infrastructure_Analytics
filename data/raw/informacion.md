# Data Raw

## Descripción

Esta carpeta está destinada a almacenar los datos originales utilizados en el proyecto **Predictive IT Infrastructure Analytics**.

Los archivos contenidos en esta ubicación deben corresponder exactamente a la versión original del dataset, sin modificaciones, transformaciones ni procesos de limpieza previos.

## Fuente de Datos

El dataset utilizado en este proyecto puede descargarse desde Kaggle:

https://www.kaggle.com/datasets/freshersstaff/it-system-performance-and-resource-metrics

## Archivo Esperado

Una vez descargado, el archivo original debe almacenarse en esta carpeta para permitir la correcta ejecución de los notebooks y scripts del proyecto.

## Consideraciones

* No modificar los archivos almacenados en esta carpeta.
* Mantener una copia íntegra de los datos originales para garantizar la trazabilidad del análisis.
* Todas las transformaciones y procesos de limpieza deben realizarse sobre copias de trabajo generadas en la carpeta `data/processed`.

## Objetivo

Preservar la fuente de datos original utilizada durante el desarrollo del proyecto, garantizando la reproducibilidad y transparencia del flujo analítico.
